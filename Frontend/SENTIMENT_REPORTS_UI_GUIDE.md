## Key Interactions

1. **Search**
   - Type ticker or company name
   - Press Enter or click "Generate Report"
   - Shows loading spinner
   - Navigates to report view

2. **Recent Searches**
   - Click any recent search
   - Instantly loads that report
   - Updates "searched at" time

3. **Horizon Selector**
   - Click any time period
   - Highlighted in lime green
   - Triggers report refresh

4. **Refresh Report**
   - Click refresh icon
   - Shows loading state
   - Updates timestamp

5. **Back Navigation**
   - Click "Back to Search"
   - Returns to home view
   - Preserves recent searches
