# Changelog

## [0.2.0] - Stock Detail View Update

### Added
- **Stock Detail View**: Transformed from portfolio view to individual stock detail page
- **StockDetailHeader Component**: Shows stock name, price, change, and market status
- **StockAbout Component**: Company information with description and key facts
- **KeyStatistics Component**: Display market cap, P/E ratio, volume, and other metrics
- **Extended Chart Periods**: Added 5Y and MAX time periods for stock charts
- **Qualcomm Financial Engine Branding**: Updated all references from Robinhood to Qualcomm
- **AI Navigation**: Added "Insights" and "Chatbot" buttons to header
- **Oracle Stock Data**: Featured Oracle (ORCL) as example stock with complete details

### Changed
- **Header Navigation**: Replaced "Rewards", "Investing", "Crypto", "Retirement" with "Insights" and "Chatbot"
- **App Layout**: Changed from portfolio dashboard to stock detail page
- **Chart Component**: Updated to support extended time periods (5Y, MAX)
- **Page Title**: Changed to "Qualcomm Financial Engine"

### Removed
- **Robinhood Legend Button**: Removed promotional button
- **Win Gold Button**: Removed from portfolio header
- **Individual Account Selector**: Replaced with stock detail view
- **Buying Power Section**: Removed as app is for insights, not trading
- **Margin Investing Card**: Removed promotional content
- **News Feed from Main View**: Focused on stock detail only

### Features
- Stock price tracking with daily changes
- Company information (CEO, employees, headquarters, founded)
- Comprehensive key statistics grid
- Multi-period interactive charting
- Stock watchlist sidebar with sparklines
- Dark theme UI maintained
- Fully responsive design
